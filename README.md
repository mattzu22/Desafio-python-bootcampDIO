# Desafio-python-bootcampDIO
Este repositório contém um desafio prático desenvolvido com o objetivo de reforçar os principais conceitos fundamentais da linguagem Python. A proposta é exercitar habilidades essenciais como:  -  Manipulação de variáveis e tipos de dados - Funções - Estruturas de dados (listas, dicionários, tuplas, conjuntos) - Boas práticas de programação
